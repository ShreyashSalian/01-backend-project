
# Backend Project

A brief description of what this project does.


## Run Locally

Clone the project

```bash
  git clone https://github.com/ShreyashSalian/01-backend-project.git
```


Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
